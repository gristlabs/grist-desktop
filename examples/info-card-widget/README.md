# Info Card Widget

A custom widget that displays the selected row as a styled card with
column mapping, persistent options, and Grist theme support.

Built as part of the [custom widgets tutorial](../../docs/custom-widgets.md#tutorial-build-an-info-card-widget).

## Install

```bash
cp -r . ~/.grist/plugins/info-card
cp /path/to/grist-plugin-api.js ~/.grist/plugins/info-card/
export GRIST_TRUST_PLUGINS=1
# Restart Grist Desktop
```
